# Starbase-Closures
SpaceX Starbase Closure Terminal App

## Installing 

Install the rich terminal library.
For windows:
```sh
python -m pip install rich
```
For linux
```sh
pip3 install rich
```
Run this command in the closure_app directory to install the application
```sh
pip install .
```
Run this command in terminal to retrieve the latest highway 4 road closures
```sh
closures
```
